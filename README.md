# php5-fpm-nginx-debug
Complete solution for debug php5.6 with fpm on nginx

```
git clone https://github.com/davidmadi/php5-fpm-nginx-debug.git
cd php5-fpm-nginx-debug
docker-compose up
```

That's it!
